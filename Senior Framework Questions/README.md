## 💡 Navigating Senior technical discussions
At the senior level, answering "What is an index?" with "It makes queries faster" is an instant rejection. You are expected to mention *B-Trees*, *Sequential Scans*, and the *write-penalty* of having too many indexes. Always drive your explanations back to memory usage, CPU constraints, and how the underlying engine operates.
